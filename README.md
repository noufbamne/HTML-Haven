# 🌐 HTML-Haven
> A comprehensive collection of HTML experiments, academic practicals, and creative web implementations.

---

## 🚀 Overview
**HTML-Haven** is a structured repository of HTML programs developed through academic coursework and hands-on experimentation.

It includes everything from **basic HTML fundamentals** to **interactive elements like audio, images, frames, and mini projects** — making it a complete practice hub for mastering HTML.

---

## 🧠 What's Inside

### 🔹 Academic Practicals
Organized HTML exercises covering core concepts as part of structured coursework.

### 🔹 Media Integration
Implementation of images, audio, and video within web pages.

### 🔹 Frames & Layouts
Experiments with framesets and multi-frame structures.

### 🔹 Tables & Structure
HTML table creation, rows, cells, and layout design.

### 🔹 Mini Projects
Small web pages and creative implementations for practical understanding.

- HTML5
- Basic CSS (where applicable)
