Top 5 Programming Languages in Computer

•C Programming

C is a general purpose, procedural computer programming language.
C in some applications might be outdated but it by no means going to extinct any time soon. It was a wide array of uses and real-world application and wil be applied in the industries for years to come.

•C++ Programming

C++ is an object oriented programming language.
The scope for C++'s similar to previously mentioned C. Its still popular and learning It will never be a bad idea. Especially C++ because it is comparatively an easier language to pick up and learn.It is also used for games as well as alongside graphical designs and 3D models.

•Java

Java is a class-based,object oriented programming language that is designed to have as few implementation dependencies as possible.
Java is general purpose language that is well for games and mobile apps development. It is also useful in enterprises level application as well as web apps. It provides structure and maintainability of code.

•Python

Python is an object oriented,high level programming language that was released way back in 1991.
Python is best way for anyone, even people with no prior experience with programming or coding languages to get started with machine learning. Python has a bright future in the devlopment field of programming languages especially in field of data visualisation, data science data analysis,etc.

•Javascript

Javascript often abbreviated as IS is a programming language that conforms to the ECMA script specification.
Javascript is a: extremely popular language. It is so widely used that it might take quite some time for another programming language to reaplace it. It mainly used in web development and artificial intelligence.
