Various options available in the target attribute

•If you set the target attribute to "blank," the link will open in a new browser/tab.

DBATU

•If you set the target attribute to "self," the link will open in the same window/tab.

Computer

•If you set the target attribute to "top," the link will open in the same window/tab.

Name

•If you set the target attribute to "parent", the link will open in the parent frame/tab.
